BlueOnyx 5207R/5208R
=====================

IMPORTANT! PLEASE READ!
------------------------

This is the development code-tree for BlueOnyx 5207R/5208R.

Until 24th April 2014 the tree had many common bits
and pieces with the 5106R/5107R/5108R code trees. Enough
so that /utils and /ui were fully interchangeable with
those from the other platforms.

But the buck stops here and today, as from now on these
code trees *will* diverge.

So please DO NOT merge in any code from the 5106R/5107R/5108R
code trees. Just DO NOT do that!

From now on the *only* merges that ARE still allowed are merges
from 5207R/5208R *downwards*. NOT *upwards*. And even then we
need to be *really* careful.

- Michael Stauber
    2014-04-24
